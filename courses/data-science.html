Master the fundamentals of data analysis, statistics, and visualization needed to build a strong AI foundation.

Course Overview

Bedrock for Data Science is a pre-requisite course for students who are looking to get into machine learning using Python. We start with the basics of Python, like data types, data structures, classes etc., and then move on to topics that cover libraries that are ML focused such as NumPy, Pandas, and Matplotlib.

Through a blend of theoretical instruction and hands-on practice, you'll learn how to collect, clean, and analyze data, as well as how to create compelling visualizations that communicate your findings effectively. The course also includes a significant project component, where students work in teams to apply their skills to real-world Python problems.

By the end of this module, students will have the necessary tools to undertake a basic course in machine learning successfully and embark on a data analytics project.

How It Works

Our teaching methodology integrates preparation, active learning, and assessment to maximize your learning experience.

Pre-Class Preparation

Before each session, students complete assigned readings and tutorials, followed by a short quiz to gauge understanding of core concepts.

Interactive Lecture

During class, instruction cycles between concise lecture segments, interactive Q&A periods, and guided hands-on exercises. This pattern repeats throughout each session to maintain engagement and reinforce learning.

Post-Class Assessment

Following each session, students complete a comprehensive quiz to solidify their understanding and identify areas for further study.

Course Curriculum

The Bedrock Data Science curriculum (Autumn-24) is a comprehensive pre-requisite course for students looking to enter the field of machine learning using Python. We start with Python fundamentals and progressively build to data science libraries and techniques.

1

Python Essentials

  • Data Types and Type Conversion
  • Variables and Operators
  • Arithmetic, Assignment, Comparison, Logical/Conditional Operators
  • Flow Control
2

Lists and Loops

  • Creating Lists, Indexing, and Slicing
  • Mutability, List Methods, and Aliasing/Copying
  • Common Sequence and Comparison Operations
  • For and While Loops
  • Mentoring Session: Iterables, Break vs Continue
3

Data Structures and Pythonic Syntax

  • Tuples and Dictionaries
  • List Comprehension
  • Dictionary Comprehension
  • Mentoring Session: Sets
4

Functions and Function Scope

  • Intuition and Definition of Functions
  • Types of Arguments: Keyword, Position, and Default
  • Unknown Number of Arguments (* and **)
  • Namespaces and Types of Scopes
  • Mentoring Session: Zip and Enumerate Functions
5

Classes

  • Class and Object Concepts
  • Attributes and Methods
  • Dunder Methods: __init__()
  • Mentoring Session: Advanced Dunder Methods (__call__, __getitem__, __str__ etc.)
6

Strings, Files and Regular Expressions

  • String Operations and Methods
  • Types of Files
  • Reading and Writing Files
  • Regular Expressions
  • Mentoring Session: Third Party Libraries
7

NumPy

  • Creating ndarray Objects
  • Dimensions and Shapes
  • NumPy Methods (zeros, ones, arrange, vstack, etc.)
  • Slicing, Accessing, Boolean Masks, and Reshaping
  • Mentoring Session: Broadcasting
8

Notations and Linear Algebra

  • Basics of Math Notations (Greek Letters, Summation, etc.)
  • Log, Exponent, and Derivatives
  • Vectors: Geometric Interpretation, Operations, and Cosine Similarity
  • Matrices: Types, Access, Multiplication, Transpose, and Inverse
  • Mentoring Session: Vector Span and Basis of Vectors
9

Probability and Statistics

  • Random Variables: Motivation and Types
  • Point Estimates: Mean, Median, Mode, Variance, and Standard Deviation
  • Histograms and Confidence Intervals
  • Common Distributions: Uniform, Binomial, and Normal
10

Pandas

  • DataFrames and Series
  • Indexing and Multi-Indexing
  • .loc() and .iloc() Methods
  • Groupby() Operations
  • Debugging with Try and Except Blocks
11

Linear Regression

  • Response vs Predictor Variables
  • True vs Statistical Models
  • Error Evaluation
  • Train and Test Sets
  • Regression Coefficients and Their Estimates
12

Data Visualization with Matplotlib

  • Exploratory Data Analysis (EDA)
  • Types of Visualizations
  • Creating Basic Plots with Matplotlib
  • Customizing and Styling Visualizations
  • Communicating Insights Through Visualization

Course Details

Everything you need to know about our comprehensive Data Science program, designed to transform beginners into job-ready data scientists.

Prerequisites

Basic programming knowledge is helpful but not required. The course includes Python fundamentals for beginners. A background in mathematics or statistics is beneficial but not essential.

  • No prior coding experience needed
  • Basic algebra and statistics concepts
  • Curiosity and commitment to learn

Format

The course is delivered through a carefully designed blend of learning methods to maximize engagement and knowledge retention:

  • Live virtual sessions with interactive Q&A
  • Self-paced learning modules with hands-on exercises
  • Real-world projects with industry datasets
  • Mentor-guided review sessions and code reviews

Tools & Technologies

Students will build a comprehensive data science toolkit with industry-standard technologies:

  • Python programming and Jupyter Notebooks
  • Data analysis with Pandas and NumPy
  • Data visualization with Matplotlib, Seaborn, and Plotly
  • Statistical analysis with SciPy and StatsModels
  • SQL fundamentals for database interactions
